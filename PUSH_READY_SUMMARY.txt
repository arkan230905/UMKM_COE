HOW IT WORKS
================================================================================

Customer Checkout Flow:
  1. Customer adds items to cart
  2. Customer clicks checkout
  3. System validates cart & stock
  4. Order created in orders table
  5. OrderItems created for each cart item
  6. Stock decremented immediately
  7. StockMovement created for audit trail
  8. OrderToSalesService syncs Order to Penjualan
  9. Penjualan created (grouped by owner if multi-owner)
  10. Payment status updated based on payment method
  11. Penjualan observer creates journals (if payment_status = 'paid')
  12. Owner sees transaction in dashboard immediately

Key Features:
  ✅ Real-Time Sync: Transactions appear immediately
  ✅ Multi-Owner Support: Separate Penjualan per owner
  ✅ Payment Tracking: Proper status tracking
  ✅ Stock Management: Consistent display
  ✅ Audit Trail: StockMovement records created
  ✅ Error Handling: Comprehensive logging
