TROUBLESHOOTING:
----------------

JIKA DATA TIDAK TERSIMPAN:
1. Buka browser console (F12)
2. Periksa error di console
3. Clear cache:
   php artisan cache:clear
   php artisan view:clear
4. Refresh halaman dengan Ctrl+F5

JIKA FOTO TIDAK TAMPIL:
1. Pastikan symbolic link sudah dibuat:
   php artisan storage:link
2. Periksa permission folder storage
3. Refresh halaman dengan Ctrl+F5

JIKA TOMBOL TAMBAH/HAPUS TIDAK BERFUNGSI:
1. Buka browser console (F12)
2. Ketik: typeof addTeamMemberRow
3. Harus return: "function"
4. Jika "undefined", clear cache dan refresh

JIKA MAPS TIDAK TAMPIL:
1. Pastikan menggunakan embed link, bukan link biasa
2. Format yang benar:
   https://www.google.com/maps/embed?pb=...
3. Cara mendapatkan embed link:
   - Buka Google Maps
   - Cari lokasi
   - Klik "Share"
   - Pilih "Embed a map"
   - Copy HTML code
   - Ambil URL dari src="..."
