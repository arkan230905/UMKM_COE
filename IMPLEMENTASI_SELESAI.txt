================================================================================
TROUBLESHOOTING
================================================================================

MASALAH: Perusahaan tidak ditemukan (404)
SOLUSI:  php artisan db:seed --class=UpdatePerusahaanSlugSeeder

MASALAH: Produk tidak muncul
SOLUSI:  Pastikan produk milik perusahaan yang benar di database

MASALAH: Route tidak ditemukan
SOLUSI:  Gunakan helper pelanggan_route() bukan route()

MASALAH: Session hilang
SOLUSI:  Verifikasi middleware set.perusahaan terdaftar di routing

================================================================================
BEST PRACTICES
================================================================================

✅ Selalu gunakan helper functions
   <a href="{{ pelanggan_route('dashboard') }}">Dashboard</a>

✅ Filter query berdasarkan perusahaan
   $produks = Produk::where('user_id', $perusahaan->user_id)->get();

✅ Gunakan middleware untuk validasi
   $perusahaan = $request->attributes->get('perusahaan');

✅ Redirect dengan perusahaan slug
   return redirect()->route('pelanggan.dashboard', ['perusahaan_slug' => perusahaan_slug()]);
