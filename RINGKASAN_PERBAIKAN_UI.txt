📋 MASALAH YANG DITEMUKAN:
   Ikon mata (toggle password visibility) tidak sejajar dengan kotak input password
   Ukuran ikon terlalu besar dan posisi tidak tepat

✅ SOLUSI YANG DITERAPKAN (UPDATE V2):

   1. WRAPPER PASSWORD FIELD
      Menggunakan: position: relative; (standard positioning)
      
   2. POSISI TOMBOL TOGGLE
      Menggunakan: right: 5px; top: 50%; transform: translateY(-50%);
      Alasan: Centering vertikal yang sempurna untuk semua ukuran layar
      
   3. UKURAN TOMBOL
      Sebelum: 32px × 32px
      Update v1: 30px × 30px
      Update v2: 28px × 28px (lebih proporsional)
      
   4. UKURAN IKON SVG
      Sebelum: 18px × 18px
      Update v1: 16px × 16px
      Update v2: 15px × 15px dengan stroke-width: 2.5 (lebih tebal dan jelas)
      
   5. PADDING INPUT
      Sebelum: padding-right: 45px;
      Sesudah: padding-right: 40px;
      
   6. HEIGHT INPUT (BARU)
      Ditambahkan: height: 35px; line-height: 35px;
      Alasan: Konsistensi tinggi input untuk alignment sempurna
