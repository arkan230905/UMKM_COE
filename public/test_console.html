🔍 Instruksi Debug Console

Langkah 1: Buka Halaman Penjualan

URL: http://127.0.0.1:8000/transaksi/penjualan/create

Action: Tekan Ctrl + F5 untuk hard refresh

Langkah 2: Buka Console

Tekan: F12

Klik tab: Console

Langkah 3: Cek Log yang Muncul

⚠️ PENTING: Screenshot console log dan kirim ke developer!

Yang HARUS muncul:

=== BARCODE SCANNER DEBUG ===
Product Data loaded: 2 products
Product Data keys (barcodes): ['8992000000001', '8992000000002']
Searchable Products loaded: 2 products
First product in searchableProducts: {id: 1, nama: 'Ayam Crispy Macdi', ...}
Products with barcode: 2

Jika TIDAK muncul atau berbeda:

  • Screenshot console log
  • Cek apakah ada error (text merah)
  • Kirim screenshot ke developer

Langkah 4: Test Ketik di Input

Ketik: 8 (satu angka)

Lihat console, harus muncul:

Input event fired! Value: 8 Length: 1
Manual typing detected, showing search results...

Langkah 5: Test Scan/Ketik Barcode

Ketik: 8992000000001

Tekan: Enter

Lihat console, harus muncul log lengkap

📸 Screenshot yang Dibutuhkan

  1. Console log saat halaman load (langkah 3)
  2. Console log saat ketik "8" (langkah 4)
  3. Console log saat scan barcode (langkah 5)
  4. Screenshot halaman lengkap

🚨 Jika Tidak Ada Log di Console

Kemungkinan masalah:

  • JavaScript error (cek tab Console untuk text merah)
  • File tidak ter-load (refresh dengan Ctrl+F5)
  • Browser cache (clear cache)

Solusi:

  1. Tekan Ctrl + Shift + Delete
  2. Clear cache
  3. Refresh halaman (Ctrl + F5)
  4. Coba lagi

✅ Jika Semua Log Muncul

Sistem berfungsi dengan baik! Lanjutkan testing sesuai checklist.
