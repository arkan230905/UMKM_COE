# INSTRUKSI URGENT - CEK BROWSER CONSOLE

## Langkah Cepat:

### 1. Jangan Tutup Halaman Error Itu!
Tetap di halaman yang menampilkan error

### 2. Tekan F12
Atau klik kanan → "Inspect" → Tab "Console"

### 3. Screenshot Console
Akan ada output debug yang saya tambahkan tadi

### 4. Kirim Screenshot Console ke Saya

---

## ATAU Jika Sudah Tertutup:

### 1. Buka Lagi Halaman Edit BOP
Klik Edit pada salah satu BOP Proses

### 2. Buka Console SEBELUM Klik Simpan
Tekan F12 → Tab Console

### 3. Isi Form
Isi beberapa komponen dengan nilai > 0

### 4. Klik "Simpan Perubahan"
Lihat output di Console

### 5. Screenshot Console
Kirim ke saya

---
