==============================================
TROUBLESHOOTING:
==============================================

Error "Access Denied":
→ Cek .env, pastikan DB_PASSWORD benar

Data lama masih ada:
→ Hapus manual: DELETE FROM coas; DELETE FROM satuans;
→ Jalankan script lagi

User baru dapat data lama:
→ Update listener
→ Clear cache: php artisan cache:clear

Masih ada "Tidak dapat diubah":
→ Cek database: SELECT user_id FROM coas LIMIT 5;
→ Pastikan user_id NOT NULL
