🔄 Contoh Skenario Penggunaan

Skenario: Membeli Ayam Potong

  1. User melihat info konversi:
    • Sub Satuan 1: Gram (1 Kilogram = 1000 Gram)
    • Sub Satuan 2: Potong (1 Kilogram = 4 Potong)
    • Sub Satuan 3: Ons (1 Kilogram = 10 Ons)
  2. User memilih "Potong" dari dropdown
  3. Sistem menampilkan: "1 Satuan Utama = 4 Potong"
  4. User input jumlah: 20 potong
  5. Sistem hitung otomatis:
    • Harga per Potong: Rp 12,500
    • Total: 20 × Rp 12,500 = Rp 250,000

🎯 Keuntungan Peningkatan

  • ✅ Informasi terpisah jelas - Data konversi ditampilkan sebagai informasi saja
  • ✅ Pilihan jelas - Dropdown terpisah untuk memilih sub satuan yang akan digunakan
  • ✅ Konversi spesifik - Menampilkan konversi khusus untuk pembelian ini
  • ✅ Visual lebih baik - Warna-warna berbeda untuk setiap jenis informasi
  • ✅ Perhitungan otomatis - Harga dan total dihitung otomatis
