## âœï¸ Cara Edit Diagram

### Mengubah Teks

1. **Double click** pada shape yang ingin diedit
2. Ketik teks baru
3. Klik di luar shape untuk selesai

### Mengubah Warna

1. **Klik** shape yang ingin diubah warnanya
2. Di panel kanan, klik **Fill** untuk warna background
3. Klik **Line** untuk warna border

### Menambah Activity Baru

1. **Drag** shape "Rectangle" dari panel kiri
2. **Drop** di posisi yang diinginkan
3. **Double click** untuk isi teks
4. **Connect** dengan arrow dari shape lain

### Menambah Decision (Diamond)

1. **Drag** shape "Diamond" dari panel kiri
2. **Drop** di posisi yang diinginkan
3. **Isi teks** pertanyaan (contoh: "Data Valid?")
4. **Connect** dengan 2 arrow (Ya dan Tidak)

### Menambah Note/Keterangan

1. **Drag** shape "Note" dari panel kiri
2. **Drop** di samping activity
3. **Isi teks** keterangan
4. **Connect** dengan garis putus-putus (dashed line)

## ðŸŽ¨ Customisasi untuk Skripsi

### Ganti Judul

1. **Klik** text box judul di atas
2. **Edit** sesuai kebutuhan:
   ```
   Activity Diagram - Proses Login
   Sistem Informasi Akuntansi UMKM
   [Nama Universitas]
   ```

### Ganti Warna Tema

**Untuk Aktivitas Pengguna:**
- Fill: `#fff2cc` (kuning muda)
- Line: `#d6b656` (kuning tua)

**Untuk Aktivitas Sistem:**
- Fill: `#dae8fc` (biru muda)
- Line: `#6c8ebf` (biru tua)

**Untuk Decision:**
- Fill: `#ffe6cc` (orange muda)
- Line: `#d79b00` (orange tua)

**Untuk Error:**
- Fill: `#f8cecc` (merah muda)
- Line: `#b85450` (merah tua)

### Tambah Logo Universitas

1. **Klik** Insert > Image
2. **Pilih** logo dari komputer
3. **Resize** dan posisikan di pojok

### Tambah Footer

1. **Drag** text box ke bawah diagram
2. **Isi** dengan:
   ```
   Dibuat oleh: [Nama Anda]
   NIM: [NIM Anda]
   Tanggal: [Tanggal]
   ```

## ðŸ”§ Tips & Tricks

### Align Shapes

1. **Select** multiple shapes (Ctrl+Click)
2. **Klik** Arrange > Align
3. **Pilih** alignment (Left, Center, Right, Top, Middle, Bottom)

### Distribute Evenly

1. **Select** multiple shapes
2. **Klik** Arrange > Distribute
3. **Pilih** Horizontally atau Vertically

### Group Shapes

1. **Select** multiple shapes
2. **Klik kanan** > Group
3. Atau tekan **Ctrl+G**

### Copy Style

1. **Klik** shape dengan style yang diinginkan
2. **Klik** Edit > Copy Style (atau Ctrl+Shift+C)
3. **Klik** shape target
4. **Klik** Edit > Paste Style (atau Ctrl+Shift+V)

### Zoom In/Out

- **Zoom In**: Ctrl + Mouse Wheel Up
- **Zoom Out**: Ctrl + Mouse Wheel Down
- **Fit to Page**: Ctrl + Shift + F

## ðŸ†˜ Troubleshooting

### File tidak bisa dibuka

**Solusi:**
- Pastikan menggunakan Draw.io versi terbaru
- Coba buka di browser: https://app.diagrams.net/
- Cek ekstensi file harus `.drawio`

### Diagram terlalu besar

**Solusi:**
- Klik View > Zoom > Fit
- Atau tekan Ctrl + Shift + F
- Atur Page Setup untuk ukuran yang lebih besar

### Teks terpotong

**Solusi:**
- Double click shape
- Resize shape dengan drag corner
- Atau enable "Word Wrap" di Format panel

### Warna tidak sesuai

**Solusi:**
- Klik shape
- Di panel kanan, atur Fill dan Line color
- Gunakan color picker atau masukkan kode hex
