METODE 1: VIA GIT (PALING MUDAH) ⭐
───────────────────────────────────────────────────────────────

Di Local (Windows):
1. git add .
2. git commit -m "Deploy desain dashboard modern"
3. git push origin main

Di Hosting (SSH):
1. ssh your-user@your-domain.com
2. cd /path/to/your/project
3. git pull origin main
4. php artisan optimize:clear

✅ SELESAI! Lanjut ke bagian VERIFIKASI di bawah.

═══════════════════════════════════════════════════════════════

METODE 2: VIA cPANEL FILE MANAGER
───────────────────────────────────────────────────────────────

STEP 1: LOGIN
1. Buka: https://your-domain.com/cpanel
2. Login dengan username & password

STEP 2: BACKUP FILE LAMA
1. Klik "File Manager"
2. Navigate ke folder project
3. Rename file lama (tambahkan .backup):
   ✓ public/css/modern-dashboard.css
   ✓ resources/views/layouts/app.blade.php
   ✓ resources/views/layouts/sidebar.blade.php
   ✓ resources/views/dashboard.blade.php

STEP 3: UPLOAD FILE BARU
Upload file-file ini dari komputer Anda:

📁 public/css/modern-dashboard.css
   → Upload ke: public/css/
   → Permission: 644

📁 resources/views/layouts/app.blade.php
   → Upload ke: resources/views/layouts/
   → Permission: 644

📁 resources/views/layouts/sidebar.blade.php
   → Upload ke: resources/views/layouts/
   → Permission: 644

📁 resources/views/dashboard.blade.php
   → Upload ke: resources/views/
   → Permission: 644

📁 app/Http/Controllers/DashboardController.php
   → Upload ke: app/Http/Controllers/
   → Permission: 644

📁 public/clear-cache.php (OPTIONAL)
   → Upload ke: public/
   → Permission: 644

STEP 4: CLEAR CACHE

Jika ada SSH/Terminal:
   php artisan cache:clear
   php artisan config:clear
   php artisan route:clear
   php artisan view:clear
   php artisan optimize:clear

Jika TIDAK ada SSH:
   1. Akses: https://your-domain.com/clear-cache.php
   2. Tunggu sampai "Success!"
   3. HAPUS file clear-cache.php (via File Manager)

⚠️ TROUBLESHOOTING CEPAT

═══════════════════════════════════════════════════════════════

PROBLEM: CSS tidak ter-load
SOLUSI:
   1. Cek file ada di: public/css/modern-dashboard.css
   2. Cek file size: ~48 KB (48,868 bytes)
   3. Cek permission: 644
   4. Upload ulang jika perlu
   5. Clear browser cache: Ctrl + Shift + Delete

───────────────────────────────────────────────────────────────

PROBLEM: Tampilan masih lama
SOLUSI:
   1. Clear Laravel cache: php artisan optimize:clear
   2. Clear browser cache: Ctrl + Shift + Delete
   3. Hard refresh: Ctrl + F5
   4. Test di Incognito: Ctrl + Shift + N
   5. Jika pakai Cloudflare: Purge cache

───────────────────────────────────────────────────────────────

PROBLEM: Sidebar masih kode mentah
SOLUSI:
   1. Clear view cache: php artisan view:clear
   2. Cek file ter-upload: sidebar.blade.php
   3. Upload ulang jika perlu
   4. Clear all cache: php artisan optimize:clear

───────────────────────────────────────────────────────────────

PROBLEM: Error 500
SOLUSI:
   1. Cek error log: storage/logs/laravel.log
   2. Fix permission: chmod -R 755 storage bootstrap/cache
   3. Clear cache: php artisan optimize:clear
