Kas Dropdown Duplicate Text Fix

❌ Before Fix (Incorrect)

The dropdown was showing duplicate "Kas" text:

  • 💵 Kas Kas Bank (Saldo: Rp 100,000,000)
  • 💵 Kas Kas (Saldo: Rp 75,000,000)
  • 💵 Kas Kas Kecil (Saldo: Rp 0)
  • 💵 Kas Kas Bank BCA (Saldo: Rp 0)

✅ After Fix (Correct)

The dropdown now shows proper account names:

  • 💵 Kas Bank (Saldo: Rp 100,000,000)
  • 💵 Kas (Saldo: Rp 75,000,000)
  • 💵 Kas Kecil (Saldo: Rp 0)
  • 💵 Kas Bank BCA (Saldo: Rp 0)

What Was Fixed

File Issue Fix
resources/views/transaksi/pembelian/create.blade.php Line 130: 💵 Kas {{ $kb->nama_akun }} Changed to: 💵 {{ $kb->nama_akun }}
resources/views/pegawai-pembelian/pembelian/create.blade.php Line 164: 💵 Kas {{ $kb->nama_akun }} Changed to: 💵 {{ $kb->nama_akun }}

Root Cause

The code was checking if the account name contains "kas" and then adding "Kas" prefix:

@if(str_contains(strtolower($kb->nama_akun), 'kas'))
    💵 Kas {{ $kb->nama_akun }}  
@else
    🏦 {{ $kb->nama_akun }}
@endif

Since account names like "Kas Bank" already contain "Kas", adding another "Kas" prefix created duplicates.
