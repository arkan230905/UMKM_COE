================================================================================
DEPLOYMENT STEPS
================================================================================

1. SSH to production
   ssh user@jobcost.eadtmanufaktur.com
   cd /var/www/html/simcost

2. Run fix command
   php artisan fix:production-user-mismatch

3. Clear caches
   php artisan optimize:clear
   php artisan cache:clear
   php artisan config:clear
   php artisan view:clear
   php artisan route:clear

4. Restart web server
   sudo systemctl restart apache2
   (or nginx + php-fpm)

5. Clear browser cache
   Ctrl+Shift+Delete, then Ctrl+F5

6. Test in browser
   http://jobcost.eadtmanufaktur.com/transaksi/penggajian/create
