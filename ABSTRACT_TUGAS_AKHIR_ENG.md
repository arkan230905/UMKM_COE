# **ABSTRACT**

*Manual production cost management causes inefficiencies in record-keeping and difficulties determining Cost of Goods Sold (COGS) for SMEs to manufacturing companies. This Final Project aims to design an integrated web application for production cost management and automatic COGS calculation using job costing methods. The development method uses waterfall approach with requirements analysis, UML system design, Laravel Framework implementation, and Black Box Testing. Tools used include PHP programming language, Laravel framework, and MySQL database. Implementation results show that application manages master data (products, raw materials, employees), calculates production costs (BBB + BTKL + BOP), provides real-time COGS dashboard, and generates comprehensive reports. The main feature is job costing capability enabling automatic calculation per job order and profitability analysis per product. This application proves effective in improving operational efficiency and production cost calculation accuracy, supporting better business decisions for SMEs and manufacturing companies in Indonesia.*

**Keywords:** production cost, cost of goods sold, job costing, web application, SME, manufacturing
